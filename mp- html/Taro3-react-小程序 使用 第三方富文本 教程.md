#### 接下来根据官方文档的微信小程序使用教程来使用就可以了

### 3.使用插件（如编辑富文本）

- 编辑文本

  ![编辑文本](./截图/文件内容.png)

- 安装依赖 重新打包

  ![打包](./截图/生成文件包.png)

```bash
# 安装依赖
# 通过 npm 安装
npm install
# 或通过 yarn 安装
yarn
```

```bash
# 生成微信包到 dist/mp-weixin
npm run build:weixin
```

### 3.重新按照上面的步骤重新将打包好的文件夹放到项目中就可以了

**注意** 使用组件时不需要导入使用的文件中是全局导入的
